Frontend not found (Angular build not packaged)

This Eden Desktop app runs the server + provides searchable logs, but it does not bundle the Angular UI.

  • Build the Angular app to: frontend/dist/eden-sim-frontend
  • Then either:
  • - Set env var EDEN_FRONTEND_PATH to that folder, or
  • - Place the folder next to the installed app at ../frontend/dist/eden-sim-frontend

Open the Logs window (Ctrl+Shift+L) to see the exact FRONTEND_PATH being used.
